# Agents Rules

1. Использовать BEM-lite: именовать блоки и элементы по BEM (`block`, `block__element`), а модификаторы добавлять только при необходимости (`block--modifier`).
2. Не усложнять селекторы: избегать глубокой вложенности и длинных цепочек.
3. Для компонентных стилей предпочитать локальную изоляцию (`scoped`), а значения выносить в CSS-переменные.
